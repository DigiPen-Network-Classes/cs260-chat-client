# CS260 Chat Client

## Building (Windows)
### Prerequisites
- Visual Studio 2022
- raylib 5.5 (via NuGet, restored automatically on build)

## Usage:

```pwsh
./CS260_ChatClient <ip> <port> <token>
```

## Chat Protocol Documentation

The Chat Protocol is described in the [Chat Server Project](https://github.com/DigiPen-Network-Classes/cs260-chat-server-solution/blob/main/README.md)

You can download the most [recent server build](https://github.com/DigiPen-Network-Classes/cs260-chat-server-solution/releases) to test locally.

A [Demo Build](https://github.com/DigiPen-Network-Classes/cs260-chat-client/releases/tag/Demo-Build) of the Chat Client (fully functional) is available 
so you can demo the program and see how it behaves.
